# Campaign Asset Generator
### Claude Orchestration · Nano Banana Pro Image Generation · MCP Pipeline

Generates complete, publication-ready social media ad images from a campaign brief. Claude acts as the creative director/agent — Nano Banana Pro (Gemini 3 Pro Image) renders the finished ads.

---

## Quick Start

```bash
export ANTHROPIC_API_KEY=sk-ant-...
export GEMINI_API_KEY=AIza...          # from aistudio.google.com/apikey

chmod +x start.sh && ./start.sh
# → http://localhost:7860
```

---

## What It Produces

Complete ad images with copy rendered directly inside them by Nano Banana Pro — not text overlaid on top, but text-in-image generated by the model. Output per platform × language:

```
exports/
  nike_tiktok_EN_916_1748123456.png
  nike_tiktok_JA_916_1748123457.png
  nike_instagram_EN_45_1748123458.png
  ...
```

## Pipeline

```
Campaign Brief + Reference Images
         │
         ▼
┌────────────────────────────────────────────────────────────┐
│               Claude Agent (claude-opus-4-5)               │
│                                                            │
│  parse_brief ──→ analyze_references (vision)              │
│                         │                                  │
│              ┌──────────┘ (style_descriptor)               │
│              │                                             │
│    For each Platform × Language:                           │
│      generate_copy                                         │
│        → score_copy (< 75? revise & re-score)             │
│          → build_image_prompt (synthesizes everything)     │
│            → generate_image ──→ Nano Banana Pro            │
│                                 (gemini-3-pro-image-preview)│
└────────────────────────────────────────────────────────────┘
         │
         ▼
    /exports/*.png
```

### The 6 MCP Tools

| Tool | Role |
|------|------|
| `parse_brief` | Structures raw campaign text into typed JSON |
| `analyze_references` | Claude vision-analyzes reference images → rich style descriptor |
| `generate_copy` | Platform-native copy per language (not translations) |
| `score_copy` | Scores 0-100 across 5 dimensions; triggers revision if < 75 |
| `build_image_prompt` | Crafts the Nano Banana Pro prompt — includes all copy text + full visual brief |
| `generate_image` | Calls `gemini-3-pro-image-preview`; accepts reference images for style anchoring |

---

## Reference Images

Upload product images or brand reference images in the UI. The agent will:
1. Vision-analyze them with Claude to extract style, lighting, color palette, composition
2. Pass that style descriptor into every `build_image_prompt` call
3. Pass the image files themselves to `generate_image` so Nano Banana Pro has direct visual context

## Environment

- Python 3.10+
- `ANTHROPIC_API_KEY` — Claude API (claude-opus-4-5 for all reasoning)
- `GEMINI_API_KEY` — Google Gemini API (gemini-3-pro-image-preview for image gen)
- No Docker. No cloud infra. One script.

## Fallback Model

If `gemini-3-pro-image-preview` is unavailable in your region, set:
```bash
export GEMINI_MODEL=gemini-2.5-flash-image
```
The server checks this env var as a fallback.
